1. Project Overview 
This document is the complete 4-day execution plan for the Career Path Generator — an AI-
powered, RAG-based career roadmapping tool. The system takes a user profile (skills, experience, 
life stage) and generates an explainable, ethically-audited career transition roadmap using Groq 
LLaMA 3, ChromaDB, and a Python RAG microservice.
